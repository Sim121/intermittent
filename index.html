Configurer Google Drive
5 étapes (~10 min) — une seule fois !
1
Va sur console.cloud.google.com et connecte-toi.
2
Crée un nouveau projet → nomme-le "Intermittent".
3
Menu Bibliothèque → cherche "Google Drive API" → Activer.
4
APIs & Services → Identifiants → "Créer" → ID client OAuth 2.0 → Type : Application Web → dans "Origines JavaScript autorisées" ajoute l'URL de ton app hébergée.
5
Copie le Client ID (finit par .apps.googleusercontent.com) → colle-le dans Réglages.
✅ Une fois connecté, toutes tes données se sauvegardent automatiquement dans un dossier Intermittent/ sur ton Drive à chaque modification.
